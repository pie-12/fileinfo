# File Inspector (GTK Version)
Chương trình C trên Linux hiển thị thông tin thuộc tính file/thư mục bằng giao diện GTK3.

## Mục tiêu
- Lấy thông tin: tên, loại, kích thước, quyền, chủ sở hữu, nhóm, thời gian chỉnh sửa.
- Duyệt đệ quy thư mục.
- Hiển thị kết quả dạng bảng (`GtkTreeView`).
